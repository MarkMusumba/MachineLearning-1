Files Included :
 Craps.py        = Python file to automate game of craps.
 Craps_op.txt = Output file obtained from the automation of game of craps.

To execute and run the code :
In Linux enviroinment :
 > Extract Craps_Automate and place them in the desired folder.
 > Goto the desired folder from terminal and type
    python Craps.py
 > The Craps_op.txt will be generated based on the automation.
 > The best stratergy worked will be displayed after every automation of the game.

The best stratergy that produces the maximum balance at the end of automation of game of craps is:
  Reverse Martingale System.
